# LaTeX 环境配置教程

## 环境

- 由于 Microsoft Word 糟糕又麻烦的公式编辑和繁琐的文档排版方式，我们不得不选择更加优雅的环境进行我们的编写。
- 目前我们使用 LaTeX 排版系统进行排版需要大家在自己电脑上配置 LaTeX 环境。

- LaTeX 系统已经广泛应用于国外的科技类文档的写作与排版中，国内的论文写作和slide制作也在使用这个系统进行

- 之后大家参加数学建模竞赛、做科研、发paper的时候还要不止一次的用LaTeX。
